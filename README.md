- [apt package manager] (https://wiki.debian.org/Apt)

Resources:
- [How to Install GNU Emacs 25.1 in Ubuntu 16.04](http://ubuntuhandbook.org/index.php/2016/09/install-gnu-emacs-25-1-in-ubuntu-16-04/)

Tips:
